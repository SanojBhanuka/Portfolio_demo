Design of 16-bit RISC Processor

Designing a 16-bit RISC processor using Verilog, this project follows the Harvard architecture, distinguishing between instruction and data memory. It incorporates essential components like a 16-bit ALU capable of executing 11 operations, a flag register for status indication, and 16 general-purpose registers. The processor supports 13 instructions with three addressing modes, emphasizing efficiency and modularity in its design.

  • Features:
    • Harvard Architecture: Enhances performance by separating instruction and data paths.
    • ALU Operations: Supports 11 arithmetic and logical operations.
    • Modular Design: Facilitates scalability and ease of debugging.
    • Instruction Set: Includes 13 instructions with three addressing modes, providing versatility for various applications.
  • Technologies Used:
    • Verilog: Hardware Description Language used for processor design.
    • Vivado Xilinx: Verilog simulation tool used for testing and validation.
    • Harvard Architecture: Separate instruction and data memory design principle.
  • Challenges:
    • Modular Design: Ensuring each module (ALU, registers, control unit) integrates seamlessly and functions correctly.
    • Instruction Set Definition: Designing and implementing a comprehensive set of 13 instructions with varying complexities.
  • Achievements:Successfully implemented a 16-bit RISC processor using Verilog, adhering to the Harvard architecture principles, which ensures separate instruction and data memory for enhanced performance. The processor supports 13 instructions with three addressing modes and integrates a 16-bit ALU capable of performing 11 arithmetic and logical operations. Through comprehensive testing and simulation using Vivado Xilinx, the project validated the correct execution of instructions, efficient data handling, and reliable flag management. This achievement not only demonstrates proficiency in hardware description and computer architecture but also contributes significantly to the field of digital system design, exemplifying the advantages of RISC architecture, such as efficient instruction execution and modular design.
