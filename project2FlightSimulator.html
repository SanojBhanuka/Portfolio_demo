Air Traffic Simulation Software

The Fast Time Flight Plan Simulator is an innovative web application designed to simulate and analyze pre-defined flight plans in real-time. It empowers users to upload CSV files containing flight data and waypoints, initiating simulations that visualize flight paths on Google Maps. The simulator integrates advanced features such as collision detection, dynamic rerouting around no-fly zones, and comprehensive flight parameter visualization. This tool bridges the gap between theoretical flight planning and practical implementation, enhancing safety and operational efficiency in air traffic management.

  • Features:
    • Real-time Simulation: Upload and simulate flight plans based on user-provided CSV files.
    • Collision Detection: Promptly identifies potential collision risks and provides detailed information.
    • Dynamic Rerouting: Automatically adjusts flight paths to avoid user-designated no-fly zones.
    • Graphical Visualization: Displays flight parameters (speeds, altitudes) against waypoints for enhanced analysis.
    • Data Export: Allows users to download detailed flight and collision information as CSV files.
  • Technologies Used:
    • Google Maps JavaScript API: Integrates with Google Maps for real-time flight path visualization.
    • MongoDB: Stores and manages flight data efficiently.
    • HTML/CSS/JavaScript: Frontend development for user interface and interaction.
  • Challenges:
    • Integration with Google Maps API: Ensuring seamless integration and efficient usage to minimize costs.
    • Real-time Data Processing: Handling and updating flight data dynamically while ensuring accuracy and performance.
    • Collision Detection Algorithm: Developing and optimizing algorithms to accurately detect potential collisions based on flight paths.
  • Achievements: A significant achievement of this project was the successful development and implementation of a robust defense mechanism for federated learning. Our FLAME (Federated Learning with Adversarial Model Evaluation) algorithm demonstrated exceptional resilience in handling scenarios with over 50% malicious clients, maintaining model integrity and thwarting adversarial attacks effectively. Particularly noteworthy was its performance in non-IID (non-identically distributed) data environments, where it mitigated data distribution challenges and improved model accuracy by 15% compared to traditional FL methods.
