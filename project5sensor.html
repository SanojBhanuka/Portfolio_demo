Object detection radar system using K-LD2 sensor

This project, developed as part of the module EE 5305 Sensors and Transducers at the University of Ruhuna, aims to enhance radar sensor technology by focusing on the K-LD2 radar transceiver. The radar sensor is designed to detect moving objects, calculate their velocity, and determine the distance from the sensor with high accuracy. Key functionalities include signal processing using Fast Fourier Transform (FFT) and Doppler Effect analysis to extract object velocity and distance data. The project culminates in presenting these results via a graphical user interface (GUI) and saving them in a text file for further analysis.

  • Features:
    • Object Detection: Detects and tracks moving objects using radar signals
    • Velocity Calculation: Utilizes Doppler Effect to calculate object velocity.
    • .Distance Measurement: Determines the distance between the sensor and detected objects.
    • Graphical User Interface (GUI): Displays real-time results including object direction and distance graphically.
    • Data Logging: Saves detection results in a text file for record-keeping and analysis.
  • Technologies Used:
    • Hardware: K-LD2 radar transceiver, Arduino Uno for signal acquisition.
    • Software: Python for signal processing and GUI development, Arduino IDE for microcontroller programming.
    • Libraries: NumPy for numerical computations, Matplotlib for data visualization, Tkinter for GUI development.
  • Challenges:
    • Signal Processing: Implementing FFT for accurate frequency domain analysis.
    • Integration: Ensuring seamless integration of Arduino, Python, and GUI components.
  • Achievements: Our project "Dynamic Object Detection Radar" represents a significant achievement in the field of sensor technology and radar systems. Developed as part of the EE 5305 Sensors and Transducers module at the University of Ruhuna, Sri Lanka, our radar sensor utilizes the K-LD2 radar transceiver to accurately detect and measure the velocity and distance of moving objects. By implementing Fast Fourier Transform for signal processing and leveraging the Doppler effect, we achieved precise object detection capabilities. The project successfully integrated Arduino for data acquisition and Python for signal analysis, demonstrating robustness in both hardware and software integration. Furthermore, our graphical user interface (GUI) provided intuitive visualization of detected objects and their parameters, enhancing usability across various applications. Overall, our project not only addressed technical challenges in radar sensor design but also showcased practical implementations crucial for real-world sensor applications.
