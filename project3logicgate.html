Logic Gate Simulation Software

The Logic Gate Circuit Simulation Software is a powerful tool designed to facilitate the creation and simulation of logic gate circuits through a user-friendly graphical interface. This software allows users to visually construct circuits using a library of logic gates, including AND, OR, NOT, XOR, and more. Once constructed, the software simulates the behavior of these circuits, providing real-time outputs based on user-defined inputs. It supports interactive editing, error detection, and the ability to save and load circuit designs for future use. Developed using Python and C++, this project was recognized among the top four entries in the Eminence 2.0 competition, showcasing its innovation and practical application in educational settings and beyond.
