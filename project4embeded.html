Assistive Communication system for Disabled

This project addresses communication challenges faced by individuals with disabilities, such as hemiparesis or speech impairments, through the development of an assistive communication system. Users interact with the system using a rotary knob and push button to select and confirm messages displayed on an LCD screen. Messages are transmitted wirelessly via a Wi-Fi module to caregivers or medical professionals, ensuring timely assistance.

  • Features:
    • User Interface: Intuitive control mechanism with a rotary knob and push button for message selection and confirmation.
    • Visual Feedback: Messages displayed on an LM016 LCD screen for user review before transmission.
    • Wireless Communication: Utilizes ESP 8266 Node MCU for reliable Wi-Fi transmission of confirmed messages.
    • Customization: Allows users to personalize messages to meet specific communication needs.
    • Accessibility: Designed to enhance communication accessibility for individuals with disabilities.
  • Technologies Used:
    • Microcontroller: ATMEGA 2560 for processing user inputs and controlling the LCD display.
    • Communication Module: ESP 8266 Node MCU for Wi-Fi communication.
    • Display: LM016 LCD screen for visual feedback and message confirmation.
    • Communication Module: ESP 8266 Node MCU for Wi-Fi communication.
    • Protocols: I2C for communication between microcontroller and LCD, USART for data transmission to Wi-Fi module.
    • Simulation: Proteus design environment for testing and validation
  • Challenges:
    • Interface Design: Ensuring the interface is intuitive and accessible for users with diverse disabilities.
    • Integration: Coordinating hardware components (microcontroller, LCD, Wi-Fi module) for seamless functionality.
    • Usability Testing: Validating system effectiveness and user satisfaction through rigorous testing with disabled individuals and caregivers.
  • Achievements: Throughout the development of this assistive communication system, significant milestones were achieved. By leveraging the ATMEGA 2560 microcontroller and ESP 8266 Node MCU communication module, we successfully created a user-friendly interface that allows individuals with disabilities to select and confirm messages via a rotary knob and push button. The integration of an LM016 LCD screen provides essential visual feedback, ensuring users can review messages before wireless transmission to caregivers or medical professionals. Our system not only enhances communication accessibility but also addresses critical challenges in interface design, hardware integration, and usability testing. This project represents a commitment to improving the quality of life for disabled individuals by bridging communication gaps and facilitating prompt assistance when needed most.
